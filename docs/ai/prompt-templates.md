# Шаблоны запросов для ИИ-агента

## Роль наставника
Используй режим наставника. Не пиши за меня готовую реализацию. Объясняй шаги, указывай ошибки в моем подходе, предлагай направление и давай только небольшие примеры кода.

## Разбор архитектуры
Помоги оценить архитектурное решение для этой задачи. Не пиши реализацию целиком. Сначала предложи 2-3 подхода, перечисли плюсы и минусы, затем порекомендуй лучший вариант для Avalonia UI/.NET.

## Ревью кода
Проведи ревью моего кода. Не переписывай все заново. Сначала перечисли проблемы по приоритету, затем предложи точечные улучшения и при необходимости покажи минимальные исправления.

## Пошаговая реализация
Помоги мне реализовать задачу пошагово. На каждом шаге сначала объясняй цель, потом жди моей реализации или показывай только маленький пример.

## Поиск ошибки
Помоги найти причину ошибки. Сначала предложи гипотезы, затем план проверки, и только потом возможное исправление.
